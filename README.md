# README

**Markdown** jest to język znaczników do formatowania tekstu. Używany w dokumentacji i README.  

### **Podstawy składni**  

#### **Nagłówki**  
Użyj `#`:  
```markdown
# Nagłówek 1  
## Nagłówek 2  
### Nagłówek 3  
```

#### **Tekst**  
- **Pogrubienie**: `**tekst**` → **tekst**  
- *Kursywa*: `*tekst*` → *tekst*  
- ***Pogrubiona kursywa***: `***tekst***`  

#### **Listy**  
- **Nienumerowana**:  
  ```markdown
  - Element 1  
  - Element 2  
  ```
- **Numerowana**:  
  ```markdown
  1. Pierwszy  
  2. Drugi  
  ```

#### **Linki i obrazy**  
- `[Tekst](https://adres.com)` → [Tekst](https://adres.com)  
- `![Opis](https://adres.com/obraz.png)`

#### **Kod i cytaty**  
- `Inline: \`kod\`` → `kod`  
- Blok kodu:  
  ```markdown
  ```python  
  print("Hello")  
  ```
  ```
- Cytat:  
  ```markdown
  > Cytowany tekst  
  ```

#### **Tabele**  
```markdown
| Kolumna 1 | Kolumna 2 |
|-----------|-----------|
| Wartość 1 | Wartość 2 |
```

#### **Linia pozioma**  
`---`
